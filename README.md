## Getting Started

***Note***: Make sure your Flutter environment is setup.
#### Installation

In the command terminal, run the following commands:

    $ git clone https://github.com/AhmadAbbas8/flutter_payment-practice.git
    $ cd flutter_payment-practice/
    $ flutter pub get
    $ flutter run
